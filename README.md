# MP4toPDF

Extract subtitle from MP4 and convert PDF

# Platform

 - macOS / Linux

# Need

Please install tools to $PATH.

 - ffmpeg
 - wkhtmltopdf
 - python3

# Install

```
pip3 install pdfkit
cd src
chmod +x mp4topdf.py
```

# How to use

```
mp4topdf.py example.mp4
```

# macOS

```
brew install ffmpeg
brew install wkhtmltopdf
```
